Aprendendo HTML

Voltar

HTML é a linguagem de marcação padrão para páginas da Web.
Com HTML você pode criar seu próprio site.
A seguir veja alguns conceitos sobre HTML

Inspetor de elementos

Voltar

É uma ferramenta disponibilizada pelo navegador para auxiliar o programador.
Com ela é possível saber como um site foi feito. É possível também fazer
alterações temporárias no site manipulando elementos da página web.

Estrutura básica do HTML

Voltar

Todos os documentos HTML devem começar com uma declaração de tipo de documento: < ! DOCTYPE html>.
O próprio documento HTML começa com < html> e termina com < / html >.
A parte visível do documento HTML está entre < body> e < / body>.

Tags

Voltar
  • As Tags são palavras reservadas com alguma função no documento HTML.
  • Essas palavras ficam entre os sinais < > .
  • Estes elementos informam ao navegador como exibir o conteúdo na página.

Atributos de Tags

Voltar
  • Todos os elementos HTML podem ter atributos
  • Os atributos fornecem informações adicionais sobre os elementos
  • Os atributos são sempre especificados na tag inicial
  • Os atributos geralmente vêm em pares nome/valor como: name="value"

Tags de Texto

Voltar

As principais Tags relacionadas a texto são:

Tags de cabeçalho
Os cabeçalhos HTML são definidos com as tags < h1 > a < h6 >.
Tag de parágrafo
Os parágrafos HTML são definidos com a tag < p >.
Tag de quebra de linha
A tag < br > Insere uma única quebra de linha

Lista ordenada e não ordenada

Voltar

As listas HTML permitem que os desenvolvedores agrupem um conjunto de itens relacionados em listas.
Podem ser de dois tipos:

Lista não ordenada
Uma lista não ordenada começa com a tag < ul >. Cada item da lista começa com a tag < li >.
Lista ordenada
Uma lista ordenada começa com a tag < ol >. Cada item da lista começa com a tag < li >.

Links

Voltar

Os links HTML são definidos com a tag < a >
O destino do link é especificado no atributo href.
